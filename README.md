## Running locally


Clone the project

```bash
  git clone https://github.com/lchenrique/tw-antd-start-react-ts.git
```

Enter the project directory

```bash
  cd tw-antd-start-react-ts
```

Install the dependencies

```bash
  npm install or yarn
```

Start the project

```bash
  npm run start or yarn start
```
